Pemrograman Web

Pertemuan 1: Pengenalan HTML

HTML (Hypertext Markup Language) adalah bahasa markup untuk membuat struktur halaman web.

<!DOCTYPE html>
<html lang="id">
<head>
<title>Halaman Web Saya</title>
</head>
<body>
<h1>Selamat Datang</h1>
<p>Ini adalah contoh paragraf</p>
</body>
</html>

Pertemuan 2: CSS Dasar

CSS digunakan untuk mendesain tampilan halaman web.

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

h1 {
  color: blue;
}

Pertemuan 3: JavaScript Dasar

JavaScript digunakan untuk menambahkan interaksi ke dalam halaman web.

<script>
function tampilPesan() {
    alert("Halo, ini pesan dari JavaScript!");
}
</script>

<button onclick="tampilPesan()">Klik Saya</button>

Pertemuan 4: Pengenalan PHP

PHP adalah bahasa pemrograman yang digunakan untuk membuat web dinamis.

<?php
echo "Halo, ini adalah halaman PHP pertama saya!";
?>

Pertemuan 5: Mengontrol Alur Program

PHP mendukung kontrol alur seperti if-else dan switch-case.

<?php
$nilai = 80;
if ($nilai >= 75) {
    echo "Anda lulus!";
} else {
    echo "Anda gagal!";
}
?>

Pertemuan 6: Perulangan dan Array

PHP mendukung perulangan dan array untuk mengelola data.

<?php
$buah = array("Apel", "Mangga", "Pisang");
foreach ($buah as $b) {
    echo $b . "<br>";
}
?>
